# 💕 Our Love

**Приложение для пар, которое превращает каждый день в совместное путешествие.**

---

## ✨ Как это работает

Our Love — это не просто трекер отношений. Это пространство, где вы и ваш партнёр создаёте что-то общее каждый день: отвечаете на вопросы, выполняете задания, делитесь настроением, сохраняете моменты и пишете послания в будущее.

> **Одна цель. Два человека. Бесконечность моментов.**

---

## 🎯 Возможности

### 💬 Вопросы дня
Каждый день — новый вопрос, который помогает узнать друг друга лучше. Ответьте на свой вопрос, и ваш партнёр увидит ваш ответ, когда ответит сам. Момент искренности, который вы создаёте вместе.

### 🎯 Задания дня
Совместные приключения каждый день. Сходить в новое кафе, посмотреть закат, приготовить ужин вместе — Our Love подскажет, чем насладиться парой.

### 📸 Наш альбом
Все ваши совместные фотографии в одном месте. Фильтрация по годам, создание коллажей — ваши воспоминания, организованные с любовью.

### 📖 Дневник
Записывайте свои мысли и впечатления. Дневник дня — это ваше личное пространство для рефлексии и обмена моментами с партнёром.

### 😊 Настроение
Делитесь своим настроением каждый день. Отслеживайте эмоциональную динамику ваших отношений и поддерживайте друг друга.

### ⏳ Капсула времени
Напишите письмо себе и своему партнёру в будущее. Запечатайте его и откройте в особенный день — годовщину,纪念日 или просто потому, что хотите.

### 🗺 Карта мест
Сохраняйте места, которые сделали ваш день особенным. Первое свидание, любимый ресторан, парк, где вы гуляли — вся ваша история в одном месте.

### 🎨 Генератор постеров
Создавайте красивые постеры из ваших совместных фотографий. Идеальный подарок или напоминание о лучших моментах.

## 🏗 Архитектура

Our Love построен на **чистой архитектуре** с разделением на слои, что обеспечивает масштабируемость, тестируемость и лёгкость поддержки.

```
Our Love/
├── DI/                    # Dependency Injection
│   ├── DIContainer.swift  # Единая точка регистрации зависимостей
│   └── AuthService.swift  # Управление аутентификацией
├── Domain/                # Бизнес-логика (не зависит от фреймворков)
│   ├── Entities/          # Сущности: User, Photo, DailyTask, Mood...
│   ├── RepositoryProtocols/ # Интерфейсы репозиториев
│   └── UseCases/          # Сценарии: HomeUseCase, AlbumUseCase...
├── Data/                  # Реализация работы с API
│   ├── API/               # APIClient, PartnerAPI
│   ├── DTOs/              # JSON-модели для API
│   ├── Mappers/           # Преобразование API → Entities
│   └── Repositories/      # Реализации репозиториев
├── ViewModels/            # Состояние UI для каждого экрана
└── Views/                 # SwiftUI-экраны с Backend-логикой
```

### Ключевые принципы

- **Чистая архитектура** — Domain слой не зависит от UI, фреймворков и сети
- **Dependency Injection** — все зависимости регистрируются в DIContainer и инжектируются
- **Repository Pattern** — абстракция над API через протоколы
- **MVVM + Backend** — ViewModels управляют состоянием, Backend-файлы инкапсулируют навигацию
- **Keychain** — токены хранятся в защищённом хранилище iOS

## 🚀 Быстрый старт

### Требования
- iOS 17.0+
- Xcode 16.4+
- Swift 5.9+

### Настройка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/salievyt/our-love-ios.git
cd "Our Love IOS"
```

2. Откройте проект в Xcode:
```bash
open "Our Love.xcodeproj"
```

3. Укажите URL вашего бэкенда в `APIClient.swift`:
```swift
private let baseURL: String = "http://localhost:8000/api/v1"
```

4. Соберите и запустите (⌘R)
